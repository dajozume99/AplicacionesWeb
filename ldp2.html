LDP: PHP

Originalmente creado por Rasmus Lerdorf en 1994.

PHP, PHP Hypertet Preprocessor, es el lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web dinámico con acceso a información almacenada en una Base de Datos, permite técnicas de programación orienteda a objetos, posee una biblioteca de funcione nativa sumamente amplia... etc.
