HTML: Breve historia

En 1980, Tim Berners-Lee, trabajador de CERN, propuso un nuevo sistema de hypertexto para compartir documentos. Con la ayuda del ingeniero de sistemas Rober Cailliau presentaron WolrdWideWeb.

En 1995, el organismo IETF (Internet Engineering Task Force) publica el estandar HTML 2.0.

En 1997, el organismo W3C (World Wide Web Consortium) publica el estandar HTML 3.2.

En 1998, se publica el estandar HTML 4.0 que añade las hojas de estilos CSS y la posibilidad de incluir pequeños prograas o scripts.

Apple, Mozilla y Opera forman WHATWG (Web Hypertext Application Technology Working Group), y ejercen presión para la salida de HTML5.

En 2014, se publica el esándar HTML5, con nuevos elementos y atributos y divs con nombres predeterminados.

Web 1.0

De carácter divulgativo, principalmente con contenido estático que lo modificaba el "Webmaster".

Web 2.0

Sustentadas bajo conexiones a internet (ADSL), mejores servidores, blogs, wikis, foros y redes sociales.

Web 1.X

Web unidireccional pero dinámica de la Web 1.0, de ahí aparecieron los llamados "portales".

Web 3.0

La web de la nube, las aplicaiones web y la web multidispositivo. Ua web ineligente que presta servicios al usuario y elimina la necesidad de sistemas operativos complejos.
