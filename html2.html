HTML: Fundamentos del lenguaje

Sintaxis

En lineas generales, sintaxis más limpia y simple. La instrucción <!DOCTYPE html> se incorpora en la primera línea para especificar el tipo de documento.

Elementos multimedia

Soporte para archivos multimedia via video y audio.

Localización geográfica

La localización de los visitantes de una página se puede conseguir con mucha más facilidad.

Almacenamiento en caché

Almacenamiento en caché de la última instancia de la página.

Comunicación CLiente-Servidor

HTML5 permite comunicaciones full duplex entre clientes y servidores.

Mecanismos Javascripts

Con la API JS Web Worker permite ejecutar JavaScript.

Compatibilidad entre navegadores

HTML5 aún no es compatible con todos los naveadores.
